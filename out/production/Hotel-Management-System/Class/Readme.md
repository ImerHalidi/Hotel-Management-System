Hotel Management System – Homework 3
Overview

This project is a Java-based Hotel Management System developed as part of Homework 3. It focuses on applying control flow structures such as if/else, switch, loops, and exception handling within an object-oriented design.

Features
Room management (StandardRoom, DeluxeRoom)
User and staff hierarchy (User, Staff, FrontDeskStaff, HousekeepingStaff)
Service handling (RoomService, LaundryService)
Booking management and validation
Discount system using codes and tiers
Detection of duplicate bookings
Input validation and custom exception handling
Technologies
Java (OOP principles)
Arrays and control flow structures
Custom exceptions
How to Run
Compile the project:
javac Class/*.java
Run the main class:
java Class.Main
Project Structure
Room hierarchy for different room types
Service hierarchy using abstraction
BookingManager for booking operations
Hotel class for system coordination
Custom exceptions for error handling
Notes
The project demonstrates real-world logic using control structures.
All 20 required tasks from the assignment are implemented and tested in the Main class.
