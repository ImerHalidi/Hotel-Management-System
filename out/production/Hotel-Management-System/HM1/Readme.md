# Hotel Management System

## Description

This is a Java-based Hotel Management System developed using Object-Oriented Programming principles.

## Features

* Room management
* Guest management
* Booking system
* Availability tracking

## Technologies Used

* Java
* OOP (Encapsulation, Classes, Objects)

## Project Structure

* Room.java
* Guest.java
* Booking.java
* Hotel.java
* Main.java

## How to Run

Compile:
javac *.java

Run:
java Main
